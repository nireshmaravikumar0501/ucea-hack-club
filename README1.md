# 🚀 UCEA Hack Club

**Innovation & Hackathon Community Portal for UCEA students**

A modern web platform to introduce students to hackathons, explore technology domains, and join a student innovation community.

---


## 🌐 Live Website
👉 https://ucea-hackclub.netlify.app/

---

## ✨ Features
- Introduction to hackathons for beginners
- Technology domain exploration (AI, Web Dev, Cybersecurity, etc.)
- Student community registration via Google Form
- Event and workshop showcase
- Responsive modern UI

---

## 🛠 Tech Stack
- Next.js
- React
- Tailwind CSS
- Framer Motion
- Netlify (Deployment)
